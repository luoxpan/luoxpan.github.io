CPP困惑系列4-一些关键字

关键字 mutable volatile
1.最常见的用法：声明类成员变量是mutable的，
用法 意味着类中的const方法可以修改这个成员 用来修饰变量，告诉编译器不要对该变量做优化，总是从内存中读取该变量真正的值。
2.用于lambda表达式中，可以直接修改以值传递的变量 该关键字无法完成线程同步。
关键字 register explicit
用法 C语言中，该关键字用来修饰局部变量，建议编译器将该值存入寄存器， 用于修饰类的构造函数，表示禁用隐式转换构造。
以获得更好的性能。但现代编译器大多会自行决定将哪些变量存入寄存器，故C++中的register其实没什么用 explicit A(int a)
register int a=2;
关键字 default delete
用法 让编译器自动生成默认构造函数，减少工作量 禁用类成员函数，一般用来在需要的时候禁用拷贝构造、移动构造、拷贝赋值，移动赋值。
class A{A()=default;} class A{ A(const A&)=delete;}
关键字 override final
放在函数声明后，用来显式的指明该函数时对基类中虚函数的重写。 放在类名后,表示该类禁止任何其他类继承它。
用法 void func() override { } 放在函数声明后，表示该函数禁止被重写。
class A final { }
void func() final { }
