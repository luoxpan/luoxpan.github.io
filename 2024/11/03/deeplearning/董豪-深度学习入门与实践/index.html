神经网络基础

单层神经网络(Single Neuron)

3 个input，一个output

\(z=x_1w_1+x_2w_2+x_3w_3+b\)

3个input，2个output

\(z_1=x_1w_{11}+x_2w_{12}+x_3w_{13}+b_1\)

\(z_2=x_1w_{21}+x_2w_{22}+x_3w_{23}+b_2\)

激活函数

上面的\(z\) 是一个线性函数，为了拟合非线性函数，引入激活函数。

输出值到[0,1]之间：sigmoid

\(y=\frac{1}{1+e^{-z}}\)

输出值到[-1,1]之间：tanh

\(y=\frac{e^z-e^{-z}}{e^z+e^{-z}}\)

输出值到任意区间：ReLU

\(y=max(0,z)\)

leaky ReLU

\(y=max(\alpha z,z)\)

多分类激活函数：softmax

\(y_i=\frac{e^{z_i}}{\sum_{j=1}^{n}e^{z_j}}\)

多层神经网络

多层神经网络就是多个单层神经网络叠加起来。

单层神经网络甚至无法完成XOR这样的任务，XOR是一个线性不可分问题，引入隐藏层，隐藏层提取出特征之后，再通过输出层完成分类，可以完成XOR任务。

更复杂的问题还需要引入更多隐藏层。

损失函数

损失函数：网络输出和预测输出之间的差异。

逻辑回归的损失函数：交叉熵损失函数

\(loss=-\frac{1}{m}\sum_{i=1}^{m}y_i\log(y'_i)+(1-y_i)\log(1-y'_i)\)

多分类的损失函数：交叉熵损失函数

\(loss=-\frac{1}{m}\sum_{i=1}^{m}\sum_{j=1}^{n}y_{ij}\log(y'_{ij})\)

\(L_p\)正则($L_p norm): 衡量一个向量的幅度

\[ \begin{aligned}&\|x\|_{p}\:=\left(\sum_{k=1}^{K}|x_{k}|^{p}\right)^{\frac{1}{p}}\\&\|x\|_{p}^{p}=\sum_{k=1}^{K}|x_{k}|^{p}\end{aligned} \]

扩展一下可以用来衡量两个向量之间的差距。

\[ \mathcal{L}_{p}=\|\boldsymbol{y}-\boldsymbol{a}\|_{p}^{p}=\sum_{k=1}^{K}|\boldsymbol{y}_{k}-\boldsymbol{a}_{k}|^{p} \]

最常用的是\(L_2\)正则损失，也就是MSE(均方差误差)。

\[ \mathcal{L}_{M S E}=\frac{1}{m} \sum_{i=1}^{m}\left(\boldsymbol{y}_{i}-\boldsymbol{a}_{i}\right)^{2} \]

\(L_1\)正则损失函数，也就是MAE(平均绝对误差)。

\[ \mathcal{L}_{M A E}=\frac{1}{m} \sum_{i=1}^{m}\left|\boldsymbol{y}_{i}-\boldsymbol{a}_{i}\right| \]

梯度下降

梯度下降：通过梯度下降算法，不断调整参数，使得损失函数最小化。

梯度下降的公式： \(w:=w-\alpha\frac{\partial\mathcal{L}}{\partial w}\)

梯度下降理论上只能找到局部最优解，但由于网络很深很大，再神经网络中一般不用担心局部最优的问题，但需要考虑鞍点问题。

反向传播

反向传播：计算损失函数对每个参数的梯度。

$$ \[\begin{aligned} &\text{1. Given:}\\ &\bullet\:a^l=f(z^l)=\frac1{1+e^{-z^l}}\\ &\bullet\:z^l=\boldsymbol{W}^{l^T}\boldsymbol{a}^{l-1}+\boldsymbol{b}^l\\ &\bullet\:\mathcal{L}=\frac12(\mathbf{y}-\boldsymbol{a}^L)^2\\ &\text{2. Then we can have the following derivatives:}\\ &\bullet\:\frac{\partial a^l}{\partial z^l}=f^{-1}(z^l)=\boldsymbol{a}^l\circ(1-\boldsymbol{a}^l)\\ &\bullet\:\frac{\partial \mathcal{L}}{\partial a^l}=(a^l-y)\\ &\bullet\frac{\partial\mathbf{z}^{l}}{\partial W^{l}}=\boldsymbol{a}^{l-1}\mathrm{and}\:\frac{\partial\mathbf{z}^{l}}{\partial\boldsymbol{b}^{l}}=1\\ &3.\text{Then the error of the output layer}\\ &\bullet\:\delta^L=\frac{\partial \mathcal{L}}{\partial z^L}=\frac{\partial \mathcal{L}}{\partial a^L}\frac{\partial a^L}{\partial z^L}=(a^L-y)\circ(a^L\circ(1-a^L))\\ &\text{4.Then the error of the other layers }l=\mathbf{1}...L-\mathbf{1}:\\ &\bullet\delta^{l}=\frac{\partial\mathcal{L}}{\partial\boldsymbol{z}^{l}}=\frac{\partial\mathcal{L}}{\partial\boldsymbol{z}^{l+1}}\frac{\partial\boldsymbol{z}^{l+1}}{\partial\boldsymbol{z}^{l}}=\delta^{l+1}\frac{\partial\boldsymbol{z}^{l+1}}{\partial\boldsymbol{z}^{l}}\\ &\quad\bullet\:z^{l+1}=W^{l+1^T}a^{l}+b^{l+1}\\ &\quad\bullet\:\frac{\partial z^{l+1}}{\partial z^{l}}=\boldsymbol{W}^{l+1^T}f^{-1}(z^{l})=\boldsymbol{W}^{l+1^T}\circ(\boldsymbol{a}^{l}\circ(1-\boldsymbol{a}^{l}))\\ &\bullet\:\delta^{l}=\frac{\partial\mathcal{L}}{\partial z^{l}}=\frac{\partial\mathcal{L}}{\partial z^{l+1}}\frac{\partial\boldsymbol{z}^{l+1}}{\partial z^{l}}=\boldsymbol{W}^{l+1^T}\delta^{l+1}\circ(\boldsymbol{a}^{l}\circ(1-\boldsymbol{a}^{l}))\\ &\text{5.The the gradients are:}\\&\bullet\:\frac{\partial\mathcal{L}}{\partial\boldsymbol{W}^{l}}=\frac{\partial\mathcal{L}}{\partial\boldsymbol{z}^{l}}\frac{\partial\boldsymbol{z}^{l}}{\partial\boldsymbol{W}^{l}}=\delta^{l}\frac{\partial\boldsymbol{z}^{l}}{\partial\boldsymbol{W}^{l}}=\delta^{l}\boldsymbol{a}^{l-1^T}\\ &\bullet\frac{\partial{\mathcal L}}{\partial b^{l}}=\frac{\partial{\mathcal L}}{\partial z^{l}}\frac{\partial\mathbf{z}^{l}}{\partial\boldsymbol{b}^{l}}=\delta^{l}\frac{\partial\mathbf{z}^{l}}{\partial\boldsymbol{b}^{l}}=\delta^{l}\\ &\text{6.Update parameters iteratively}\\&W^{l}:=W^{l}-\alpha\:\frac{\partial\mathcal{L}}{\partial\boldsymbol{W}^{l}}\quad b^{l}:=b^{l}-\alpha\:\frac{\partial\mathcal{L}}{\partial\boldsymbol{b}^{l}} \end{aligned}\]

$$

为什么sigmoid函数会导致梯度消失？

\[ \delta^{l}=\frac{\partial\mathcal{L}}{\partial\mathbf{z}^{l+1}}=\frac{\partial\mathcal{L}}{\partial\mathbf{z}^{l+1}}\frac{\partial\mathbf{z}^{l+1}}{\partial\mathbf{z}^{l}}=\delta^{l+1}\boldsymbol{W}^{l+1^T}\circ(a^{l}\circ(1-a^{l})) \]

\(a^l\circ(1-a^l)\)是sigmoid函数的导数，当\(a^l\)接近0或1时，\(a^l\circ(1-a^l)\)接近0，且这种现象在深层网络中会进一步放大，导致梯度消失。

优化算法

mini-batch 随机梯度下降（SGD）

每次迭代使用所有的数据来计算损失值和梯度，代价太大，速度慢。

每次迭代随机选取一个mini-batch来计算损失值，更新梯度。

自适应学习率

学习率是超参数，需要手动设置，且难以选择。

自适应学习率算法会根据梯度的大小自动调整学习率，如Adam、RMSprop等。

防止过拟合

解决过拟合问题的最直观的方法是增加数据集，但很多时候没有这么多训练数据，因此有了数据增强的方法。

正则化是一种防止过拟合的方法，通过在损失函数中添加一个正则化项来限制模型的复杂度。这种方法也称为权重衰减。

正则化：\(\mathcal{L}_{total}=\mathcal{L}+\lambda\|W\|\)

L1正则化：\(\|W\|_1=\lambda\sum_{i=1}^{n}\left|w_{i}\right|\)

L2正则化：\(\|W\|_2^2=\lambda\sum_{i=1}^{n}w_{i}^{2}\)

Dropout是一种防止过拟合的方法，通过在训练过程中随机丢弃一部分神经元来实现的。

Dropout可以看作是一种集成学习方法，每次迭代都会训练一个不同的模型，然后将这些模型的预测结果进行平均。

Dropout可以防止过拟合，但会增加训练时间。该方法基本只在全连接层中使用。

批量归一化：作用于层的内部，先将层的输出进行归一化，然后才进入激活层。注意再使用批量归一化的层的上一层不要加bias参数。

卷积神经网络（CNN）

动机

为什么需要为图像设计专门的神经网络？

因为一个图像上的数据点很多，如果使用全连接层，参数太多，计算量太大。

同时人们发现，图像的局部区域是高度相关的，因此可以使用卷积层来提取局部特征。

卷积层

卷积层的作用是提取局部特征。

卷积层的基本思想是使用一个小的滑动窗口（称为卷积核）在输入数据上滑动，然后计算窗口内的数据与卷积核的乘积之和。

卷积核的大小是超参数，通常为3x3或5x5。

卷积层可以提取图像的边缘、纹理等特征。

池化层

池化层的作用是进行特征聚合，减少参数数量。

池化层的基本思想是使用一个小的滑动窗口（称为池化核）在输入数据上滑动，然后取窗口内的最大值或平均值。

池化层可以减少特征的数量，同时保留重要的特征。

卷积神经网络的结构

卷积神经网络通常由多个卷积层、池化层和全连接层组成。

卷积层用于提取图像的局部特征，池化层用于进行特征聚合，全连接层用于进行分类。

经典的卷积神经网络结构包括LeNet、AlexNet、VGGNet、GoogLeNet和ResNet等。

反卷积（转置卷积）

反卷积是一种卷积操作的逆操作，用于将特征图恢复到原始大小。

计算机视觉算法

目标检测

如何评价检测结果的好坏？

IoU（Intersection over Union）表示预测框和真实框的交集与并集的比值。

目标检测的准确率和召回率是两个常用的评价指标。

准确率（Precision）表示预测为正样本中实际为正样本的比例。

TP表示真正例，FP表示假正例。TP 有东西且识别出东西的概率，FP 没有东西但识别出东西的概率。

\(Precision=\frac{\text{TP}}{\text{TP}+\text{FP}}\)

召回率（Recall）表示实际为正样本中被预测为正样本的比例。

\(\text{Recall}=\frac{\text{TP}}{\text{TP}+\text{FN}}\)

FN 表示有东西但没识别出东西的概率。

mAP（Mean Average Precision）是目标检测任务中常用的评价指标，它考虑了每个类别的准确率和召回率。

R-CNN

R-CNN（Region-based Convolutional Neural Networks）是一种经典的目标检测算法。

R-CNN的基本思想是使用选择性搜索算法（Selective Search）在图像中提取候选区域，然后对这些候选区域进行卷积神经网络的特征提取，最后使用支持向量机（SVM）进行分类。

步骤： 1. 使用选择性搜索算法在图像中找出2000个候选区域。 2. 将每个候选区域缩放调整为指定大小 3. 对缩放后的候选区域使用VGG进行卷积神经网络的特征提取。 4. 使用SVM对特征进行分类，得到每个候选区域的类别和边框 5. 一张图里没有2000个物品，所以还需要非极大值抑制（NMS）来去除重叠的候选区域。

R-CNN有许多不足： 1. 搜索算法很慢 2. 无论什么大小的候选区域都缩放为指定大小，影响了特征提取的效果 3. 在每个候选区域上使用VGG进行特征提取，计算量很大 4. 并非端到端训练

SPPNet

SPPNet（Spatial Pyramid Pooling Network）是对R-CNN的改进，它解决了R-CNN的2、3问题。

提出了空间金字塔池化，把不同尺寸的候选区域映射到固定大小的特征向量，从而避免了缩放候选区域的问题。

将整张图作为输入，而不是每个候选区域都执行VGG网络。

SPP 的缺点： 1. 依然使用选择算法，速度慢 2. 依然没有使用端到端训练

Fast R-CNN

Fast R-CNN是对R-CNN的改进，它将候选区域的特征提取和分类过程合并为一个网络，从而减少了计算量。

  1. 提出了ROI Pooling层，他是空间金字塔池化的简化版本，只有一个池化尺度
  2. 将分类和边框回归合并为一个神经网络，而不是使用传统机器学习SVM进行分类

Fast R-CNN的缺点： 1. 依然使用选择算法

Faster R-CNN

Faster R-CNN是对Fast R-CNN的改进，它将候选区域的生成和特征提取过程合并为一个网络，从而进一步减少了计算量。

  1. 提出了Region Proposal Network（RPN），用于生成候选区域
  2. 完全的端到端训练

YOLO

YOLO（You Only Look Once）是一种实时目标检测算法。

前面就是一堆卷积，关键在于理解最后的输出(7x7x30)。

7x7是网格大小，表示把一个图片分为7x7的网格。每个格子有一个30维的向量，这个向量里面就是这个网格的预测结果。

这30个参数分别表示什么？

其中10个分为两个部分，表示以这个网格为中心检测到的两个物体的边框坐标和置信度。可以看出它的一个假设是每个小框里面有两个物体。

剩余20个参数预测这个边框属于哪一个类别。

注意： * 一个cell预测的两个边框共用一个类别预测，在训练是会选取与标签IoU更大的一个边框负责回归该真实物体框，在测试时会选取置信度更高的一个边框，另一个被舍弃，因此7x9个cell最多只能预测49个物体。

最后YOLO算法还需要执行NMS算法去除重叠区域。

YOLO 的缺陷：

检测小物体效果不好，因为一个网格只能检测两个物体，所以小物体很容易被忽略。

YOLO v2

预定义anchor，预定义很多边框尺寸，这些尺寸几乎可以覆盖各种类型物体的尺寸。

改进： 1. 将7x7个cell增加为13x13 2. 为每个cell预定义6个anchors 3. 每个anchor有其对应的物体类别预测和边框回归预测

图像分割

FCN （全卷积网络）

全卷积网络（Fully Convolutional Network，FCN）是一种用于图像分割的神经网络。将图像分割分为Encoder和Decoder两部分。Encoder使用卷积层和池化层提取信息，Decoder使用反卷积层将信息恢复到原始大小。

FCN的细节效果很差，因为编码过程中丢失了太多信息，这些信息没有被反卷积层获取。

U-Net

U-Net的主要特点是使用了跳跃连接（skip connection），将编码器和解码器之间的特征进行连接，从而保留了更多的图像信息。

后续的很多研究都通过加入跳跃连接来提升细节处的分割效果。
